# Nexora — Digital Studio Portfolio

Современный интерактивный веб-сайт и клиентский портал для цифровой студии **Nexora**.

Проект создан как визуальный и функциональный **портфолио-сайт**, чтобы показать красивый современный интерфейс, интерактивные элементы, анимации и возможности React-приложений. Главная идея дизайна — объединение органического человеческого творчества **Carbon** и машинной точности **Silicon**.

Nexora использует атмосферу цифровой лаборатории: контрастная типографика, зернистые текстуры, ретро-эффекты, dithered halftone noise и плавные физические анимации.

---

## О проекте

Nexora — это концепт сайта цифровой студии, которая занимается брендингом, дизайном, интерфейсами и созданием современных веб-продуктов.

Сайт не ограничивается обычной презентацией услуг. Пользователь может взаимодействовать с элементами интерфейса: собирать бренд-кит, рассчитывать стоимость проекта, просматривать статьи, загружать файлы в клиентский кабинет и отслеживать этапы работы.

Основная задача проекта — показать уровень frontend-разработки, UI/UX-дизайна и работы с интерактивными компонентами.

---

## Основные возможности

### 1. Brand Kit Experiment

Интерактивный конфигуратор бренда, расположенный в разделе **Try Now**.

Пользователь может:

* ввести название своего бренда;
* выбрать сферу деятельности;
* выбрать визуальный стиль;
* получить подходящую цветовую палитру;
* увидеть подходящие шрифтовые пары;
* получить автоматически созданный слоган;
* копировать HEX-коды цветов в буфер обмена одним кликом.

Доступные стили бренда:

* Minimalist Slate
* Warm Organic
* Bold Tech
* Cyber Punk

Каждый выбранный стиль меняет визуальное направление будущего бренда: цвета, настроение, шрифты и подачу.

---

### 2. Before / After Wireframe Slider

Интерактивный слайдер расположен в разделе **Capabilities**.

Он позволяет перетягивать разделительную линию и сравнивать два состояния дизайна:

* технический wireframe-макет;
* готовый современный интерфейс.

Слева пользователь видит чертёжную сетку с техническими обозначениями, а справа — финальный высококонтрастный дизайн с визуальными эффектами, текстурами и атмосферой Nexora.

Этот блок показывает путь проекта от идеи и структуры до готового продукта.

---

### 3. Project Estimator

Интерактивный калькулятор стоимости проекта находится в разделе **Pricing**.

Пользователь может самостоятельно настроить параметры будущего сайта:

* выбрать количество экранов;
* настроить детализацию графики;
* выбрать уровень сложности анимаций;
* увидеть обновление стоимости в реальном времени.

После расчёта пользователь может нажать кнопку **«Зафиксировать цену»**. Рассчитанная стоимость сохраняется в текущей сессии, чтобы пользователь мог вернуться к ней позже.

---

### 4. Nexora Insights

Раздел блога и новостей студии.

Пользователь может просматривать материалы по категориям:

* Strategy
* Design
* AI
* Craft

Возможности раздела:

* фильтрация статей без перезагрузки страницы;
* интерактивные карточки публикаций;
* модальные окна для комфортного чтения;
* адаптивный размер текста;
* кнопки «Мне нравится»;
* сохранение состояния лайков во время работы с сайтом.

---

### 5. Client Portal

Закрытая рабочая зона клиента, которая имитирует реальный кабинет заказчика.

Внутри пользователь может работать с материалами проекта, общаться с менеджером и отслеживать процесс разработки.

#### Asset Handoff Portal

Интерактивный загрузчик файлов.

Поддерживает:

* drag-and-drop загрузку файлов;
* ручной выбор файлов;
* отображение списка загруженных материалов;
* удаление файлов из списка.

Этот блок создан как пример интерфейса для передачи логотипов, фотографий, текстов, брендбуков и других материалов от клиента студии.

#### Онлайн-чат с архитектором

Интерактивный чат с представителем студии.

Пользователь может отправлять сообщения и имитировать рабочую переписку с менеджером или дизайнером проекта.

#### Milestones Tracker

Мини-трекер этапов разработки.

Позволяет отслеживать статус проекта, например:

* Исследование
* Стратегия
* Дизайн
* Разработка
* Тестирование
* Запуск

## Установка и запуск проекта

Склонируйте репозиторий:

```bash
git clone https://github.com/your-username/nexora.git
```

Перейдите в папку проекта:

```bash
cd nexora
```

Установите зависимости:

```bash
npm install
```

Запустите проект в режиме разработки:

```bash
npm run dev
```

После запуска откройте ссылку, которую покажет Vite в терминале.

Обычно проект будет доступен по адресу:

```bash
http://localhost:5173
```

---

## Сборка проекта

Для создания production-версии выполните команду:

```bash
npm run build
```

Для предварительного просмотра готовой сборки:

```bash
npm run preview
```
